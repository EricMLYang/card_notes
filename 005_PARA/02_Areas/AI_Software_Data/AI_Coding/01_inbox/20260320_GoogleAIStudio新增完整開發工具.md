王者 Google 開出生態系整合大絕

Google 剛在 AI Studio 裡塞進了一套完整的 vibe coding 環境，整合了自家的 Antigravity 程式開發 agent 和 Firebase 後端基礎設施。表面上是功能更新，背後是 Google 把三個產品串成一條開發管線的整合動作。

AI Studio 原本是 Google 的瀏覽器端 AI 測試工具，現在加入了全端開發能力。用自然語言描述 app 需求，Antigravity agent 自動處理前端、後端、多檔案管理，Firebase 負責資料庫（Firestore）和使用者驗證（Firebase Auth），API 金鑰也由系統安全儲存。

Google 展示的 demo 是一個叫 GeoSeeker 的即時多人遊戲，整合 Google Maps API，有即時多人狀態同步和羅盤導航邏輯，全部從瀏覽器裡用對話建出來的。

Antigravity 去年 11 月才公開預覽，短短四個月已經在 SWE-bench 跑出 76.2% 的成績，排在所有公開基準測試的程式開發 agent 前段班。免費使用期間還支援 Claude Opus 4.6 等多款模型，目前在 AI 開發工具的使用率穩定維持在 10% 左右。

Google 真正在做的是把三層產品堆成垂直整合，AI Studio 負責原型測試，Antigravity 負責自主開發，Firebase 負責後端基礎設施，形成一條從「想法到部署」的完整管線，全部在 Google 生態系內完成。

Cursor 賣 IDE，Replit 賣雲端環境，Claude Code 賣終端機體驗，Google 的打法是免費送全套，靠 Firebase 的用量計費回收。

目前 AI 開發工具的格局是 Claude Code 在使用率領先，Cursor 和 Windsurf 在功能排名領先，Google 用「免費 + 生態系整合」的角度切入。這一次更新把 AI Studio 從模型測試工具，變成可以直接出產品的開發平台。

開發者未來可能不再需要學怎麼寫程式碼，但一定需要基礎設施，這正是 Google 最不缺的東西。 📡
