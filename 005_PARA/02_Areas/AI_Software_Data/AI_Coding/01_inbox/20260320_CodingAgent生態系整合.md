王者 Google 開出生態系整合大絕

Google 把 AI Studio 升級成完整的 vibe coding 平台，整合了 Antigravity 程式開發 agent 和 Firebase 後端。用自然語言就能建出有資料庫、登入驗證、API 串接的全端 app，demo 是一個即時多人遊戲。

Antigravity 去年 11 月公開預覽，四個月就在 SWE-bench 跑到 76.2%，免費還能用 Claude Opus 4.6。Google 的策略是把 AI Studio、Antigravity、Firebase 三層堆成垂直整合，從想法到部署不用離開 Google 生態系。

Cursor 賣 IDE，Replit 賣雲端，Claude Code 賣終端機，Google 免費送全套，靠 Firebase 用量回收。開發者未來可能不需要學寫程式碼，但一定需要基礎設施。
