## 摘要

HBR 提出「Agent Manager」這個正在企業中成形的新角色：負責編排 AI Agent 的學習、協作與執行，確保 AI 與人類工作者的安全整合。類比於「軟體革命催生了 PM，AI 革命正在催生 Agent Manager」。文章定義了六個核心能力，並提出組織架構建議。

## 關鍵段落

**Agent Manager 的六大能力：**
> "AI 操作素養（理解 agent 機制與故障診斷）、功能深度（業務領域專業知識）、系統思維（跨工作流多 agent 編排）、變革韌性（快速測試部署學習循環）、提示工藝（塑造 agent 行為的語言邏輯）、混合工作流設計（無縫的機器-人類流程）"

**最有力的商業案例：**
> "整合 agents 將團隊能力從 30 天內預約 150 個會議，提升到單週超過 350 個會議。"

**組織架構建議：**
> "Agent Manager 應歸屬業務部門（客戶成功、銷售管理），而非集中式 IT。將 agent manager 與 AI 工程師配對的組織，擴展速度更快且更受信任。"

**最關鍵的角色定義：**
> "最優秀的 agent manager 來自理解被自動化業務流程的角色。Zach Stauber 的背景——音頻製作與對話設計——說明領域專業知識和『真誠好奇心』比技術資歷更重要。"
