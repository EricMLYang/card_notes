# AWS Marketplace



目前多為單點產品 (EX: 快速報表產品、快速分析) 或是像 Databricks 這種 SaaS 的全流程通用產品（ Databricks 在 AWS Marketplace 也滿受歡迎的）



**市場格局更新** AWS Marketplace 最近（2025年7月）推出了「AI Agents & Tools」新類別，這是一個重要的市場變化，表明 AI 代理和工具正成為新的熱門領域

**Databricks 和 MongoDB Atlas 確實是熱門產品** AWS Marketplace 主頁確實將 Databricks Data Intelligence Platform 和 MongoDB Atlas 列為頂級產品

這些頂級產品都採用 SaaS 交付模式



**Syntasa 確實以顧問服務形式上架** Syntasa 確實在 AWS Marketplace 上以「Quick Start Plus Package」形式提供，這是一個由 ABSC 作為系統整合商提供的專業服務包，包含部署、訓練和後續支援

您關於「即買即用、針對特定中型業務場景的數據AI方案較少」的觀察有一定道理，但需要考慮：
