Mockdown.design：用 Markdown 語法快速繪製視覺化架構圖
​
發現一個超酷的工具！
大家現在玩 Vibe Coding 應該一直聽到 md md md（Markdown），這個 Markdown 除了寫 README 說明書還可以幹嘛？
​
Markdown（.md）是用來寫文件的輕量標記語言，但 Mockdown 它把 Markdown 的概念延伸到視覺設計領域。用類似 Markdown 的語法來快速畫出視覺化的架構圖和線框圖！
而且左側有蠻多種繪製方式，有認真與不認真的（威）
畫完的內容右側會轉換成樹狀圖結構，還可以點擊回到你繪製的圖像，有夠厲害。
​
■ 工具核心概念
➀ 用文字語法取代拖拉介面：像寫程式一樣「寫」出設計圖
➁ 最後輸出成 .md 檔案：方便版本控制和協作
➂ 視覺化呈現：自動生成帶有設計感的圖形
➃ 快速原型工具：比傳統設計工具更適合快速發想
