# 三層進化：從 GenAI 圖表建議到 Agent

## 類型
Agent 發展路徑

## 概念
Agent 進化三階段：(1) 第一層：GenAI 圖表建議——AI 分析數據後建議適合的圖表類型；(2) 第二層：自動化執行——AI 不只建議，還自動生成圖表；(3) 第三層：Agent 主動行動——AI 主動監控數據變化、發現異常、產生報告並通知相關人員。關鍵轉變：從「被動回應」到「主動預測」，從「單一任務」到「端到端流程」，從「人機交互」到「機器自主」。

## 重要性
提供 AI 產品進化的清晰路線圖。

## 邊界
每一層的信任門檻不同，需要逐步建立。
