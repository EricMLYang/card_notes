# 3-三層進化：從 GenAI 圖表建議到 Agent

**類型**：Model

## 概念
AI 應用進化可分三層：L1 建議層（生成洞見）、L2 執行層（完成任務）、L3 代理層（持續負責結果）。大多數產品停在 L1，因為只輸出建議不承擔後果；要進入 L2/L3，必須補齊工具調用、狀態記憶、驗證與回饋迴圈。

## 重要性
它可用來判斷產品成熟度，避免把「會說」誤判成「會做」。

## 邊界/反例
高風險領域（金融、醫療）即使進到 L3 仍需人工核決；全自動代理不代表可免責。

## 知識鉤子
可與 `#Agent 產品95%失敗原因` 串成從 demo 到 production 的能力缺口地圖。
