# 3-人才斷層危機：初階職位被跳過，如何養成資深人才？

**類型**：Warning

## 概念

王伯達觀察 Claude for Financial Services 後的警示：**過去的工作流程**：分析師做分析 → 助理排版 → 主管審核 → 修改。**現在的工作流程**：Claude 一次產出 → 主管審核 → 完成。中間那些「初階分析師」和「助理」的角色被跳過了。問題是：如果這些工作都不用做了，還需要初階分析師嗎？如果不需要初階分析師，怎麼培養資深分析師？這是 AI 帶來的隱性問題——不是「取代」而是「跳過」，造成人才養成的斷層。

## 重要性

這是組織需要思考的長期風險——短期省人力，長期沒人才。

## 邊界/反例

某些公司可能刻意保留初階職位作為人才培育管道；或發展新的培育模式（如讓初階員工從「做事」轉為「審核 AI」）。
