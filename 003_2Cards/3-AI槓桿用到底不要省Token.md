# AI 槓桿用到底，不要省 Token

- **類型**：Principle
- **概念**：現階段個體戶用 AI 算 CP 值很不明智。AI 廠商在搶市場、瘋狂補貼，你買一塊他送你好幾塊。本地端 LLM 與雲端頂級模型有明顯差距，省那一點錢換來更差產出、更慢速度、維護環境時間成本，不划算。月薪十萬的人，一小時值五六百，一個月花一千塊訂閱 AI，省下兩小時就回本。正確策略是：趁 token 便宜的補貼期，把 token 換成真金白銀——用它接更多案子、產出更多內容、完成更多專案、學會更多技能。當補貼退場、價格上漲時，你已經累積了經驗差距，知道怎麼下 prompt、什麼任務適合丟給 AI、怎麼整合進工作流程。
- **重要性**：這是時間窗口策略——AI 補貼期不會永遠存在，現在的累積決定未來的競爭力。
- **邊界/反例**：有機密敏感內容或 NSFW 需求的必須 local。要提供服務給客戶的商業場景，可控性和穩定性是合理考量。
