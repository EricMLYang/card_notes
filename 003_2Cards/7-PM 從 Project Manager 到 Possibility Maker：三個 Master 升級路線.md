# 7-PM 從 Project Manager 到 Possibility Maker：三個 Master 升級路線

**類型**：Model

## 概念

Agent 已能自主擬定計劃、拆 To-do、準備測試、交出成果——PM 過去做的事 AI 做得挺好的。AI 時代 PM 的工作重點從「執行」變成「想法推動」：Prompt Master（會用 AI）→ Prototype Master（能做出東西）→ Proposal Master（能說服人買單）。最後一步是真正的瓶頸：你晚上指揮 10 隻 Agent 覺得很威，白天回到人間 1 個人都說服不了。提案才能破局——向老闆/客戶/合作單位獲取資源（錢、人力），不提案 Possibility 就永遠 impossible。更進一步，PM 要成為 Positioning Maker——看懂局勢，在對的位置注入想法。

## 重要性

對任何想推動變革的人（不限 PM 職稱）的角色自我檢驗——你卡在哪個 Master？

## 邊界/反例

在高度技術導向的組織中，Proposal 的說服力可能取決於技術實力而非提案技巧。三個 Master 不一定是線性升級，有些場景可以跳級。
