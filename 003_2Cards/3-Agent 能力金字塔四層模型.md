# 3-Agent 能力金字塔四層模型

**類型**：Model

## 概念

從 Agent 失敗模式歸納出的能力層級金字塔：(1) **基本工具使用與規劃**：拆解任務、選工具、正確傳參數、不跑掉（GPT-4o 等中階模型卡在這層）；(2) **適應力**：計畫碰壁時能調整（如搜尋沒結果會改用其他方式）；(3) **接地能力**：保持在當前脈絡，不亂編 ID 或瞎掰事實；(4) **常識推理**：分隔 GPT-5 和人類的關鍵（如判斷「包裹已收到」是退貨不是取消訂單）。2025 年是 agent 開始穩定行動、可以討論常識推理能力的一年。

## 重要性

這是評估 Agent 產品成熟度的框架——知道自己的 Agent 卡在哪一層，才能對症下藥。

## 邊界/反例

這個金字塔基於客服任務測試，不同領域（如 coding、研究）的能力層級可能不同。此外，不同模型的「尖刺狀智力」意味著某模型可能在第四層某些任務表現好但第二層表現差。
