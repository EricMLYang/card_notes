## 標題
從 Prompt 技巧到 Context 資產：AI 時代的獨特性來源

## 類型
Principle

## 概念
AI 時代「問好問題」的價值正在被商品化（Prompt 範本、最佳實踐），真正的競爭優勢轉向**提供獨特 Context 的能力**。

**Context 的不可複製性來源**：
- 長期累積的價值觀與判斷偏好
- 特定領域的反芻見解與失敗經驗
- 個人獨特的跨領域組合

這讓相同的 AI 工具在不同人手中產出差異化結果——不是因為 Prompt 技巧更好，而是因為輸入的情境建立在難以複製的個人知識資產上。

**關鍵配套能力：鑑賞與質疑**
AI 產出速度快但品質不穩定，需要透過「迭代篩選循環」：
1. 提供 Context → AI 大量產出
2. 質疑、過濾、收斂 → 找出關鍵方向
3. 再次迭代 → 熬煮出真正有價值的問題與答案

這是一種設計思維過程：不是一次問對，而是透過多輪對話逐步逼近核心洞見。

## 重要性
揭示 AI 協作的長期競爭力來源：不是學會最新 Prompt 技巧，而是累積難以複製的個人 Context 資產。

## 邊界/反例
- 新手階段仍需學習基礎 Prompt 技巧作為入門
- 標準化任務（如格式轉換）不需要獨特 Context，通用 Prompt 即可
- 過度強調個人 Context 可能忽視團隊知識共享的價值
