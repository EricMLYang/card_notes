# 14-高開發成本曾是天然過濾器：AI 降低門檻後的四大組織副作用

**類型**：Warning

## 概念

AI 讓寫 code 變便宜，但高開發成本曾是天然品質過濾器——強迫組織篩選好點子。過濾器消失後的四大副作用：(1) 組織用驚人速度產出大量無人要的垃圾功能；(2) 多數人用 AI 在 2 小時混完 8 小時工作，節省的時間歸私人生活（「合法薪水小偷」進化）；(3) 真正頂尖的工程師被大量 slop code 壓垮，快速崩潰離職——隱藏的高品質人才流失成本；(4) 就算 code 寫再快，流程、部門政治、官僚主義不變，效率提升就是假議題。CFO 只盯著每人每月 $200 AI 費用問為什麼，看不到組織層面的慢性崩潰。

## 重要性

這是導入 AI 工具到傳統組織前必須正視的組織副作用清單——效率工具不改變組織瓶頸，反而可能加速劣化。

## 邊界/反例

這些副作用在 AI-native、工程文化強的組織（如 Codex 團隊）中較不明顯，問題核心是組織文化和管理而非 AI 工具本身。有品質把關機制（如 TDD + Interface/Test）的團隊可以緩解大部分副作用。
