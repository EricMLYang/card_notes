# 從黎曼幾何到 Uber：基礎研究的曲折長線

- **類型**：Model
- **概念**：知識累積是人類歷史中最曲折的故事。從黎曼幾何到 Uber：(1) 19 世紀黎曼研究「彎曲空間的幾何」，純理論無應用；(2) 50 年後愛因斯坦用它建構廣義相對論；(3) GPS 必須納入相對論修正，否則定位誤差達數公里；(4) Uber 乘車調度必須仰賴精準 GPS。手機上按一個按鈕，背後是數千年知識累積。基礎研究對經濟活動的影響是「曲折」的，可能多年後才發酵，但一旦應用，其影響尺度極大——往往是某種經濟活動「成」或「不成」的臨門一腳，具有 Leontief Function 的「弱鏈特性」。
- **重要性**：解釋為何基礎研究看似無用卻關鍵，支持長期視野的研發投資思維。
- **邊界/反例**：並非所有基礎研究都會有應用——許多理論可能永遠停留在「純學術」。需要耐心等待，但也需要多元佈局而非押注單一方向。
