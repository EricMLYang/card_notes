# 3-Prompt 素養與輸出品質相關性 0.92——AI 是差距擴大器

**類型**：Warning

## 概念

Anthropic 報告揭露殘酷數據：**Prompt 的教育水準與輸出品質的相關性高達 0.92**。這打破了「AI 能拉平智力差距」的幻想——事實上，AI 是擴大器。資深軟體架構師因深厚領域知識，能寫出技術邏輯嚴密的 Prompt，引導 AI 產出神級代碼；初學者不知如何精確描述問題，只得到平庸甚至錯誤答案。「你的學歷和專業知識並沒有作廢，它們變成了你的指揮權。你必須具備辨別什麼是好答案的能力，這需要比以前更扎實的基礎。」

## 重要性

這是判斷「AI 對我是紅利還是威脅」的關鍵——你的領域知識決定 AI 輸出上限。

## 邊界/反例

某些標準化任務（如翻譯、摘要）對 Prompt 品質要求較低，差距較小；但涉及專業知識的任務差距會極端放大。
