# 4-AI Coding Agent 的五種協作模式

**類型**：Model

## 概念

基於 Anthropic 內部團隊使用 Claude Code 的真實案例，AI Coding 有五種協作模式：(1) **截圖式除錯**：把 K8s 儀表板截圖餵給 Claude，它能引導找出 Pod IP 耗盡等問題，過去需網路專家的難題幾分鐘解決；(2) **非技術人員數據操作**：財務團隊用白話文寫需求，Claude 轉成自動化流程；(3) **代碼庫導航**：新人把 Claude.md 丟給 Claude 問「我想做某任務要看哪些檔案」，快速理解依賴關係；(4) **非同步原型開發**：啟用 auto-accept mode 讓 Claude 自主編寫 80%，人類最後微調（Vim 模式就這樣開發 70%）；(5) **同步核心功能開發**：涉及商業邏輯時提供詳細指令、即時監控品質。

## 重要性

這是 AI Coding Agent 的真實應用全景圖，定義了「什麼場景用什麼模式」。

## 邊界/反例

複雜或非常規架構（如 Kafka + CQRS + gRPC 微服務）仍會讓 AI 困惑；核心功能仍需緊密同步監控。
