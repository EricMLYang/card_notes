# 4-Karpathy：Vibe Coding 重塑軟體業——程式變得免費可塑用完即丟

**類型**：Model

## 概念

Karpathy 2025 年度回顧：「Vibe coding」讓程式設計不再嚴格保留給專業人員，任何人都可以做。這是「Power to the people」的又一例：與以往所有技術形成鮮明對比，**普通人從 LLM 獲得的好處遠多於專業人員、企業和政府**。但 vibe coding 不僅賦能普通人接觸程式設計，也讓專業人員能寫出更多否則永遠不會寫的程式。Karpathy 用 vibe coding 寫了自己的高效 BPE tokenizer（用 Rust），甚至 vibe coded 整個一次性 App 只為了找一個 bug。「程式突然變得免費、短暫、可塑、用完即丟。」

## 重要性

這是理解「軟體的本質正在改變」的核心洞見——程式從昂貴資產變成廉價工具。

## 邊界/反例

核心商業邏輯、需要長期維護的系統仍需要專業品質；但大量「膠水代碼」和一次性工具確實可以 vibe coding。
