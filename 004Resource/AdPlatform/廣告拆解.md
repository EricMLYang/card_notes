# 廣告拆解



## **不同實體場域的「可分析元素」（特徵工程清單）**

### **通用核心（所有場域都應紀錄）**

1. **品牌可辨識度**：是否出現「獨特資產」（Logo、包裝外形、口號、角色等），以及**出場時間是否夠早**（有／無、在第幾秒/第幾格）。獨特資產是提升記憶與連結的捷徑，是創意效果的關鍵驅動之一。 

2. **訊息長度**：主訊息字數（OOH/戶外多以**7 字以內**為常見經驗法則）。 

3. **視覺階層**：是否有**單一焦點**、標題/主視覺/Logo 的層級清楚（是/否，或 1–5 分量表）。 

4. **可讀性**：字體大小 vs 預期觀看距離、**高對比配色**（對比是否達建議組合/比例）、動態版位是否避免過亮白底。 

5. **CTA（行動呼籲）**：有無（掃碼／加 LINE／到貨架），**最多 1 個 CTA**。 

6. **動態元素（DOOH）**：是否使用動態、剪接速度、**版位循環長度（常見 6 秒訊息窗）**、第一畫面資訊是否即懂。 

7. **情緒/調性**：歡樂、專業、安心、幽默（可多選），與品牌定位是否一致（是/否）。 

8. **情境化/動態投放**：是否依**地點/天氣/時段**自動換版（是/否；觸發條件）。 

9. **環境阻礙**：反光、雜亂背景、觀看角度（面對/斜視）、預估視距（m）與**停留/可視時間**（秒）。 

10. **量化指標**：實測/估算的**注意力或停留**（dwell、eyes-on、注意時間）。可用電腦視覺量測 dwell、in-view、attention。 





### **依場域補充（把「可視時間」與「可讀量」差異化）**

- **便利商店（走道/貨架前）**：視距近（1–3m）、移動快 → 短文案、強對比、單一焦點、就近 CTA（帶路到貨架/價促）。若在櫃台排隊，可放稍多資訊。 

- **藥妝/藥局（候位/取藥區）**：**等待時間長（約 10–12 分鐘級）** → 可放教育型內容（症狀/用藥提醒）、較長文案與分段敘述。 

- **販賣機**：**即刻決策** → 超短訊息 + 大圖示價格/口味，強 CTA（按鍵/掃碼），畫面首秒完成溝通。 

- **博物館/展場**：**長停留與近距離閱讀** → 標題＋100 字以內說明、合宜字級與閱讀距離規範、可讀性與無障礙優先。

# **用 AI 生成實體廣告的「最順手流程」（短鏈、可反覆迭代）**

1. **簡報/任務卡（Brief）**

   目標（銷售/到店/教育）、受眾、**場域與規格**（尺寸、像素、安全區、檔案格式/秒數）、法規限制、可用**品牌資產**（Logo/包裝/口號）、量測方式（dwell/銷售/回憶）。 

2. **場域對照檢核**

   - 硬規則: **字數上限、對比門檻、字級/視距、秒數**

3. **概念與腳本雛形**

   先出 3–5 個方向（每個方向都**含主標≤7字**、單一焦點、對應 CTA），DOOH 則寫出「前 1–2 秒就能懂」的訊息腳本。 

4. **生成模板（Prompt 模組化）**

   - 文案 Prompt：\[場域\] + \[主標≤7字\] + \[調性\] + \[CTA\] + \[品牌資產必入鏡\]

   - 視覺 Prompt：\[主體\] + \[獨特資產\] + \[高對比色組合\] + \[單一焦點構圖\] + \[版位尺寸比例\]

   - 變體策略：配色、構圖、標題同義改寫、Logo 位置/大小梯度。 

   

5. **自動規格化與合規檢查**

   一鍵輸出各版位尺寸/長寬比；檢查**對比**、**字數/字級**、**CTA≤1**、**循環秒數**、**安全區**。 

6. **注意力預測 & 小樣測試**

   以注意力/視線熱區或 A/B 快測挑選前 1–2 名；**注意力是創意效果的重要前因**，但要看「有意義的注意」而非只看曝光。 

7. **上線與量測**

   收集 **dwell / in-view / attention** 與銷售、回憶度，形成「**特徵 → 表現**」的迭代資料，持續優化 Prompt 與版面。
