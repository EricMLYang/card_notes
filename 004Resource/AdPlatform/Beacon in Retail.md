## \[ Beacon 運作方式 \]

- **廣播訊號(Broadcasting Signals)**：

   - Beacon 裝置會持續不斷地以低功耗藍牙（Bluetooth Low Energy, BLE）技術廣播一個無線電訊號，這個訊號包含少量的資料包，稱為 **Advertisement Packet**

   - 主要包含 3 個重要的標識符（Identifier）：

      - **UUID (Universally Unique Identifier)：** 一個 128 位元的全球唯一識別碼，區分不同的品牌或應用程式。

      - **Major：** 16 位元數字，用於區分同一品牌或應用下的不同區域（例如，分店）。

      - **Minor：** 16 位元數字，用於進一步細分區域（例如，特定貨架或櫃位）

   - 每隔 100\~1000 毫秒發送一次廣播訊號，這個間隔可以調整，來平衡功耗與偵測速度

- **手機接收**：效費者手機需開啟藍牙並安裝相關 App 才能接收訊號

   - **藍牙需開啟：** 必須開啟藍牙才能偵測 Beacon 訊號

   - **App 支援：** 需安裝支援 Beacon 的 App（例如商家自家 App）才能處理訊號

- **資料傳送**：手機收訊號後，傳送資料到應用 Server，進行應用動作

   - App 偵測 Beacon 訊號，根據 UUID、Major、Minor 進行比對，觸發特定行為

   - **後端可能處理：**

      - **資料儲存：** 將訊息儲存在資料庫中，方便後續分析

      - **即時分析：** 分析用戶行為並觸發行銷推播（優惠券或新品通知）

      - **自動化行銷：** 結合其他系統(EX: CRM) ，進行更個人化推播



## \[ 應用概述 \]

### 場域案例

- **百貨公司與購物中心**

   - **Mall of America（美國）：** 使用 Beacon 導覽顧客，並提供個人化的購物建議

   - **Harrods（英國）：** 根據顧客的購物行為推送 VIP 專屬折扣

- **連鎖超市與便利商店**

   - **Carrefour（法國）：** 使用 Beacon 分析顧客購物路徑，優化商品擺放與促銷策略

   - **7-Eleven（日本）：** 提供行動支付優惠通知，提升顧客黏著度

- **品牌專賣店（如服飾、鞋店）**

   - **Uniqlo（優衣庫）：** Beacon 輔助分析顧客在店內的動線，優化店面設計

   - **Adidas（愛迪達）：** 顧客靠近特定鞋款，自動推送該款式的產品影片和推薦搭配

- **餐飲業（咖啡廳、速食店、餐廳）**

   - **Starbucks（星巴克）：** Beacon 用於行動支付和個人化優惠推播，提升消費體驗

   - **McDonald’s（麥當勞）：** 推送附近門市的限時優惠，增加即時消費機會

- **零售藥局與美妝店**

   - **Sephora（絲芙蘭）：** Beacon 提供產品試用指南和個人化化妝建議

   - **Walgreens（沃爾格林）：** 推送藥品補貨提醒和健康管理建議，提升顧客忠誠度

- **娛樂與活動場所（展覽、博物館、主題樂園）**

   - **The MET（大都會博物館）：** 使用 Beacon 提供自助導覽服務，提升參觀體驗

   - **Disneyland（迪士尼樂園）：** 結合 App 提供即時遊樂設施排隊時間，提升遊客滿意度

### **應用**

- 消費者體驗提升：

   - **室內導航：** Beacon 引導顧客找到特定品牌專櫃或洗手間，改善大型商場的導覽體驗

   - **快速結帳：** Beacon 協助快速辨識顧客身分，縮短會員結帳時間

   - **互動試衣體驗：** 當顧客進入試衣間，Beacon 通知銷售人員提供協助

   - **健康管理提醒：** Beacon 結合健康 App，提醒顧客補充常備藥品

   - **智能導覽：** Beacon 結合語音導覽系統，提供多語言的自動化解說

   - **互動體驗：** 參觀者靠近特定展區時，解鎖專屬的互動內容或AR效果

- **即時推播優惠：** 

   - 顧客走到特定樓層或店鋪前，推送限時折扣或新品資訊

   - 當顧客在特定區域停留超過一定時間，推送相關產品推薦

   - 顧客靠近展示櫃，App 自動顯示商品的詳細資訊、顏色、尺寸庫存等

   - 顧客進店時即收到當日的特惠活動通知

- **會員精準行銷：** 

   - 根據顧客的停留時間和區域，推送個性化的優惠券

   - 當顧客接近冷藏區，自動推播當週特價商品

   - 當 VIP 顧客靠近，推送專屬折扣或生日優惠

- 營運管理：

   - **庫存管理：** Beacon 結合員工的手機 App，提醒補貨需求

   - **排隊管理：** 提供實時排隊狀況，並在叫號時發送通知

   - **動線分析：** 分析人流分布，優化展覽配置與安全管理



## \[ **Beacon 成熟度 \]**

Beacon 技術自 2013 年由 Apple 推出 iBeacon 協議以來，已經過多年的發展。儘管技術本身相對成熟，但其應用情境仍在不斷演進，具體成熟度可以從以下幾個層面來分析：

### **1\. 技術層面：已成熟**

- **穩定性：**BLE（藍牙低功耗）已廣泛應用於智慧穿戴、IoT 設備等，技術穩定且功耗低

- **跨平台支援：**\
   支援 iOS 和 Android，並且許多開源開發工具（如 Estimote SDK、[Kontakt.io](Kontakt.io) API）降低了開發門檻。

- **硬體成本下降：**裝置已高度商品化，價格便宜、電池壽命長，可長達 2-3 年

###  **2\. 應用層面：部分場景成熟，部分仍在探索**

- **成熟應用場景：**

   - **零售行銷：** 百貨公司、便利商店等已大量部署，用於即時推播與會員互動（如 Carrefour、Starbucks）

   - **智慧導覽：** 博物館、機場、展覽場、樂園等，用於室內定位與導覽

   - **智慧辦公：** Beacon 與辦公室 IoT 結合，實現無卡進出與智能環控

- **仍在探索的場景：**

   - **智慧交通：** Beacon 結合 V2X（車聯網）的潛力尚在試驗階段

   - **醫療場景：** 病患追蹤與設備管理的應用尚未大規模普及

   - **廣告精準投放：** 結合戶外數位看板進行行人精準行銷，還需克服隱私與數據整合挑戰



## \[  **Beacon 的未來發展趨勢 \]**

### **挑戰**

1. **用戶依賴性：**需用戶開啟藍牙並安裝特定 App，摩擦較高

2. **隱私與法規：**GDPR 等隱私法規對個人化推播和數據收集提出了更嚴格的要求

3. **定位精度限制：**室內定位精度約 1\~3 公尺，難滿足高精度場景需求

4. **技術競爭：**Wi-Fi RTT、UWB（超寬頻）等新興技術在定位精度上有更大優勢

### **趨勢**

隨著 IoT、5G、AI 等發展，Beacon 將拓展應用場景，尤其是在智慧城市與精準行銷領域

- **與 IoT 深度整合：** 

   - Beacon 與智慧燈光、環控系統整合，提升智慧空間管理

   - 與智慧建築、智慧城市的 IoT 設備協同工作，擴展應用場景

- **5G 與邊緣計算：** 

   - 結合更快的數據傳輸，實現即時行為分析與推播

   - 結合 AI 進行用戶行為分析，實現更智慧的個人化行銷

- **無需 App 模式：** 透過 PWA（Progressive Web App）或 Web Beacon 技術，降低摩擦

- **多模態定位：** 結合 GPS、Wi-Fi、UWB 等多種技術，提升定位精度與可靠性
