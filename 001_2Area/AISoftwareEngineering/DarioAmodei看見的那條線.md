【Dario Amodei 看見的那條線】

⌬ 十五年來，那條線從沒變過

Anthropic CEO Dario Amodei 觀察 AI 領域十五年了。他在今年達沃斯論壇（World Economic Forum）接受 WSJ 訪談時，講了一個我覺得非常精準的框架：AI 技術能力沿著一條平滑的指數曲線穩定成長，跟 Moore’s Law 很像。但公眾情緒每三到六個月就翻轉一次，在「AI 改變一切」和「全是泡沫」之間劇烈擺盪。

「這是認知現象，不是技術現實。」他看到的是 intelligence 的 Moore‘s Law。模型的認知能力每隔幾個月都在顯著提升，但外界看到的只有熱潮和恐慌的交替。媒體的極端擺盪，掩蓋了底層那條持續穩定的進展曲線。

我自己從半導體到雲端到 AI，在產業裡默默觀察，身處其中看到的進步是連續的、漸進的，但外部報導永遠是斷裂的。一下子說 AI 要取代所有人，一下子又說泡沫要破了。可能人在現場，時間比較緩慢移動吧（？

⌬ 高成長搭配高失業，歷史上沒見過

Amodei 丟出了一個讓我停下來想很久的預測：AI 的經濟特徵可能是「5-10% 的 GDP 成長伴隨 10% 的失業率」。這兩個數字同時出現，邏輯上完全合理，但歷史上從未發生過。

過去我們直覺認為高 GDP 成長就代表充沛的就業機會。但 AI 可能打破這個連結。軟體可能將變得極度便宜甚至接近免費，一場會議可能只花幾分錢就能做出一個臨時解方的應用軟體。而整個持續數十年的職涯方向可能因此消失。Amodei 說社會「完全沒有意識到即將發生什麼」。

我自己從台積製程整合轉到雲端系統架構，再到產品與技術整合，每一次挑戰「整個職涯方向被重塑」的親身體驗。但這次 AI 帶來的規模顯然大得多，不是換一個產業的問題，而是所有知識工作流程與時間感，同時被重新定義。

⌬ 「我不再寫程式了」

Anthropic 內部的工程主管告訴 Amodei，他們已經不再自己寫程式了，全部交給 Opus 處理，自己只做編輯。Claude Code 在開發者社群造成了突破時刻。Amodei 形容：進步一直是漸進的，但 Opus 4.5 到了一個臨界點，「突然被注意到了」。

更讓我意外的是非技術使用者的反應。他們發現 Claude Code 不只能寫程式，還能整理待辦事項、規劃專案、處理資訊。需求強烈到他們願意跟 command line 搏鬥（認真的，command line 對非技術使用者來說是多痛苦的介面）。Amodei 稱這是 unmet demand。Anthropic 花了兩週用 Claude Code 本身打造了非技術版本 Claude Cowork，推出幾天內指標就是過去所有產品的四倍。

我從去年過年後開始玩 Claude Code，2025 年 5 月在 AWS Summit Hong Kong 的演講主題就是「Reinventing Programming - How AI Transforms Enterprise Coding」。這個觀察我可以直接驗證：當 AI 寫程式的品質過了某個門檻，工作模式會徹底改變。不是效率提升 20% 的那種改變，是根本不同的做法。

⌬ 科學家選了一條不一樣的路

Amodei 指出 AI 產業由兩種人主導。科學家背景的領導者（他自己和 DeepMind 的 Demis Hassabis）有長期思考技術影響的傳統，「從一開始就是為了創造對世界有價值的東西」。社群媒體世代的創業家則不同，selection effects 不同，跟消費者互動甚至操控的方式也不同。

Anthropic 因此刻意選擇企業市場而非消費者市場。他的邏輯很直接：「不要讓自己需要對抗自己的商業動機。」消費者市場的壓力會把你推向 engagement 最大化、製造低品質內容、依賴廣告。企業市場是直接賣有價值的產品，不需要靠十億免費使用者的注意力戰爭來變現。

在 AWS 生態系玩耍了十幾年，我理解「為客戶提供價值」和「靠注意力變現」是完全不同的遊戲。Anthropic 選企業市場，某種程度上跟 AWS 的思路類似：先把基礎建設做好，讓客戶拿去創造價值。

⌬ 當 AI 能做所有的「工作」

Amodei 聊到教育，短期問題是用 AI 作弊，但長期問題更根本：在 AI 重塑一切的世界裡，我們到底該教什麼？他認為教育應該回到早期的概念，塑造品格、豐富個人，而不是純粹為了經濟目的。

他也丟出了一個讓人警惕的場景：一個「第零世界國家」（zeroth world）可能正在形成。矽谷大約 1,000 萬人加上散布全球的 300 萬人，形成去耦的經濟體，GDP 成長率可能達到 50%，但跟全球經濟脫節。「意識型態將無法在技術現實面前存活。」他預測這些討論終將超越黨派，成為所有人的共識。

身為一個一直在學的人，從 PKM 到 拆解整合工作流程 到 AI，我越來越覺得學習的價值不只是職業技能。當 AI 能做越來越多「工作」而且越做越快的時候，我是誰、我怎麼思考、我關心什麼，這些反而變成了真正重要的事。
