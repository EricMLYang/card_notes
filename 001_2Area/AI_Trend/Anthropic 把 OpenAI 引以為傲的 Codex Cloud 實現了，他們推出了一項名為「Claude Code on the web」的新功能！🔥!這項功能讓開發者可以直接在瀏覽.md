Anthropic 把 OpenAI 引以為傲的 Codex Cloud 實現了，他們推出了一項名為「Claude Code on the web」的新功能！🔥

這項功能讓開發者可以直接在瀏覽器中將程式開發任務指派給 Claude，無需開啟終端機。它可以同時執行多個任務，並在獨立的沙箱環境中運行，確保程式碼的安全性。

不只這樣，丟去雲端開發的變更，可以隨時抓到本機更新本地的檔案！💪

對於處理積壓的錯誤、日常修復或進行平行開發工作來說，這是一個相當理想的工具。目前這項功能是為 Pro 和 Max 使用者提供的研究預覽版。👨‍💻￼
