## **AWS 顧問**

# 🚀 AWS Select Tier Partner 最終執行規劃

## 📋 **執行摘要**

- **目標**：取得 AWS Select Tier Partner 資格

- **時程**：4-5 個月

- **人力需求**：4 人團隊

- **總成本**：約 US$3,000

---

## 👥 **人員配置與認證規劃**

### **技術組（2人）**

**人員A（技術主管）& 人員B（工程師）**

- ✅ AWS Technical Professional Accreditation（免費線上課程）

- ✅ AWS Cloud Practitioner 認證（US$100/人）

- ✅ AWS Solutions Architect Associate 認證（US$150/人）

### **商務組（2人）**

**人員C（業務經理）& 人員D（專案經理）**

- ✅ AWS Business Professional Accreditation（免費線上課程）

### **認證成本明細**

- Cloud Practitioner：US$100 × 2人 = **$200**

- Solutions Architect Associate：US$150 × 2人 = **$300**

- **認證總成本：US$500**

---

## 📅 **4階段執行時程表**

### **第1個月：啟動準備**

- [ ] 註冊 AWS Partner Network（免費）

- [ ] 指定 Alliance Lead（通常為人員A）

- [ ] 啟動 4 人認證培訓計畫

- [ ] 規劃內部 AWS 專案識別

### **第2-3個月：認證取得**

- [ ] 完成所有 Accreditations（4人）

- [ ] 技術組考取 Cloud Practitioner（2人）

- [ ] 技術組考取 Associate 認證（2人）

- [ ] 開始登記 ACE Pipeline 商機

### **第3-4個月：商業案例建立**

- [ ] 登記 3 筆 Launched Opportunities

- [ ] 確保總 MRR ≥ US$1,500

- [ ] 收集 1 個公開可引用客戶案例

- [ ] 準備客戶推薦文件

### **第4-5個月：申請與審核**

- [ ] 繳交年費 US$2,500

- [ ] 遞交 Select Tier 申請

- [ ] 通過審核，取得正式資格

- [ ] 建立對外 Partner 形象

---

## 💰 **成本結構**

| 項目 | 金額 | 備註 | 
|---|---|---|
| APN 註冊 | 免費 | 基本會員 | 
| 年費 | US$2,500 | Select Tier 必繳 | 
| 認證考試費 | US$500 | 4人共6張證照 | 
| **總成本** | **US$3,000** | 第一年 | 

---

## 📈 **預期效益**

### **直接效益**

- 獲得 AWS Select Tier Partner 認證

- 可使用 AWS Partner 標誌增加客戶信任

- 獲得 US$3,500 AWS 推廣點數（年費回本）

### **間接效益**

- 進入 AWS Partner Central 生態系

- 取得行銷資源與技術支援

- 為後續升級 Advanced Tier 建立基礎

---

## 🎯 **關鍵成功要素**

### **必達指標**

1. **4人認證達標**（技術2人各3張、商務2人各1張）

2. **3筆商機登記**（總MRR ≥ $1,500）

3. **1個客戶推薦**（公開可引用）

4. **年費按時繳交**（US$2,500）

### **風險管控**

- **Plan B**：準備6人團隊防止認證失敗

- **商機備案**：識別5個潛在專案確保3個成功

- **客戶關係**：提早與既有客戶溝通推薦事宜

---

## 📞 **下步行動**

### **立即執行（本週）**

1. 確認 4 人團隊成員與角色分工

2. 註冊 AWS Partner Network 帳號

3. 制定詳細認證時程表

### **內部提案重點**

- **投資回報**：年費 $2,500 → 獲得 $3,500 點數

- **市場定位**：成為官方認證 AWS 合作夥伴  

- **團隊能力**：提升 4 人 AWS 技術實力

- **業務機會**：開啟 AWS 生態圈合作商機
