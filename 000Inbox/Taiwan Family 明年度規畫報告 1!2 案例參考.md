零售媒體網絡（Retail Media Network, RMN）的概念，其核心邏輯

。將實體店鋪資產徹底「媒體化」

。將實體流量與交易數據進行「貨幣化」

。零售商角色從商品銷售者（Merchant），轉為擁有受眾的媒體發布者（Publisher）

。RMN 模式下，店鋪的價值則取決於其作為「廣告接觸點（Touchpoint）」的觸及率（Reach）與轉化力。





一、先給結論 S4M 角色:

我們比較像  伊藤忠商 + 宇匯知識科技 Bridgewell 的混合角色，

透過多家場域客戶合作，達到規模效果，

並且希望透過店家有個安全機制提供自家數據，

讓我們用這個價值點去跟線上、其他領域數據談合作，

目前困難點會是要如何先達到規模。





二、日本 FamilyMart 案例

利用全日本16,000 家店鋪網絡，結合高密度數位看板（FamilyMart Vision）、AI 攝影機，以及線上線下會員數據，構建出全日本最大規模的實體零售媒體平台



FamilyMart 目標不僅是廣告收入，更希望透過數位化的媒體體驗來提升顧客的店內體驗，進而反哺核心的零售業務。



結盟:

FamilyMar + 

有商流與物流能力的伊藤忠商事 +

有龐大會員數據與通信技術的 NTT Docomo +

有數位廣告操作與創意能力的 CyberAgent 進行深度結盟





FamilyMart  + 伊藤忠商 = 株式會社 Gate One（Gate One Corp.）

。主任務: 將店內空間轉為媒體版位。包括數位看板、安裝維護、內容編排以及廣告時段銷售。

。媒體產品化**：**將螢幕整合成標準化的媒體產品「FamilyMart Vision」，具備了全國聯播與區域定向能力，方便大型品牌廣告主進行採購。

。內容策略： 避免引起顧客的反感，會開發具吸引力的非廣告內容（如新聞、氣象、娛樂短片），以維持媒體的觀看率與顧客好感度。





 FamilyMart + 伊藤忠商事 + NTT Docomo + CyberAgent  = 「株式會社 Data One（Data One Corp.）」

**。**跨域數據整合： 將 FamilyMart 的線下購買數據與 NTT Docomo 的 dPoint 會員數據（擁有超過 9,000 萬會員庫，涵蓋性別、年齡、居住地、興趣標籤等）進行 ID 層級的串聯。

。線上線下歸因**：** CyberAgent 的廣告技術能追蹤用戶在數位平台（如 YouTube、Instagram）上的廣告瀏覽行為，並將其與隨後的線下 FamilyMart 購買行為進行匹配。讓廣告主能夠清晰地計算出線上廣告投資報酬率（ROAS）。

。產品開發： 開發基於數據的廣告產品，例如「購買者重定向（Purchaser Retargeting）」或「類似受眾擴展（Look-alike Modeling）」，向廣告主提供成效分析報告。

。重要優勢: 

    -FamilyMart 提供「買了什麼（Purchase）」

    -NTT Docomo 提供「他是誰（Identity）」與「他在哪（Location）」

    -CyberAgent 數據： 提供「他對什麼感興趣（Interest）」的數位行為

三、台灣數據案例

1\.禾多移動 AviviD / LRMN (來客行銷大聯盟)

1-1 核心定位： 

。以「One ID」為核心的跨產業數據聯盟，試圖建立龐大生態系，將不同來源的數據串接在同一個 ID 下，形成類似「實體＋數位」的全通路 RMN。



1-2.數據整合範疇 ：

。媒體數據： 500+ 家媒體（內容農場、新聞站）， 6,600 萬訂閱用戶的瀏覽行為標籤。

。實體零售數據： 宣稱 5,000+ 實體門市（含便利商店、百貨等）

。電信數據： 利用手機訊號分析地理位置（居住地、工作地）與生活型態。

。發票數據： 透過發票載具獲取實際消費金額與品項，驗證「消費力」



1-3.技術手段：

。One ID： 這是核心技術，將述破碎數據歸戶到同一個去識別化的 ID 上。

。AI 自動貼標： 對用戶進行興趣與行為的自動分類。



1-4. 商業模式：

。打造類似 RMN 效果 ，讓沒大數據的中小電商能享有 RMN 等級的精準投放。

。賣廣告版位，強調提供「品牌聲量」與「受眾輪廓分析」



2\. 宇匯知識科技 Bridgewell

2-1. 核心定位： 

。「Data Clean Room (資料無塵室)」為核心的技術賦能者

。他們角色較像是「安全的數據交換中心」與「AI 廣告技術提供商」

。重點在於讓零售商敢於把數據拿出來變現。



2-2 數據整合範疇：

。歷史數據深： 有超過 15 年在地消費者數據經驗，累積 3,600+ 產品與興趣標籤。

。零售第一方數據對接： 協助綜合零售、母嬰、生活百貨等產業，將其會員數據與宇匯的廣告數據庫對接。



2-3 技術手段：

。Data Clean Room (資料無塵室)： 提供加密、去識別化的環境，讓零售商（擁有交易數據）與品牌主（擁有廣告預算）進行數據 Mapping，雙方都看不到對方的原始個資，但能針對交集到的受眾進行精準投放。

。CVRi (轉換距離預測)： 獨家 AI 技術，預測消費者「距離購買還有多遠」，來決定出價策略。



2-4 商業模式：

。RMN 技術合作夥伴： 幫助零售商建立自己的 RMN 流量變現機制

。DSP/SSP 廣告生態： 提供從數據管理到廣告投放的一條龍技術服務
