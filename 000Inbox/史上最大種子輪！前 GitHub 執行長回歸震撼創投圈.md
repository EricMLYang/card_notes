▋史上最大種子輪！前 GitHub 執行長回歸震撼創投圈

前 GitHub 執行長 Thomas Dohmke 昨天投下了一顆震撼彈。

他的新創公司 Entire 在種子輪就狂募了 6,000 萬美元，估值高達 3 億美元，直接打破開發工具領域的歷史紀錄。

為什麼資本市場這麼買單？

因為這家公司號稱可以解決「AI 寫程式太快」所帶來的災難。這可是當前 AI coding 最大的痛點

Dohmke 看到了一個重要關鍵：我們沿用幾十年的 Git 版本控制系統，根本處理不了 AI 時代的洪流。

現在 AI Agents 生成程式碼的速度遠超人類理解能力，導致開源專案被大量低品質的「AI 廢料」淹沒。

Entire 的核心產品 Checkpoints 就是要解決這個痛點，它能把 AI 寫程式的「提示詞」和「思考脈絡」跟代碼綁定。

這不只是單純的 debug，這是讓人類工程師在 AI 海嘯中真的能看懂「機器在想什麼」
