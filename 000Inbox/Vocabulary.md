# Vocabulary

- stitch : needs to stitch together data from multiple fabrics

-  accommodate, heterogenous : A large data platform needs to accommodate heterogenous data storage. By *heterogenous data storage*, we mean data spread across multiple data fabrics.

- overkill : Azure Data Explorer with its high-performance indexing and caching capabilities might be overkill

- upstream : we can perform analysis across multiple datasets coming from different upstream teams

- propagated : We would have to track the data flow of a single dataset to make sure a fix is propagated correctly.

- Anatomy : **Anatomy of a data platform**

- unprecedented : the amount of data generated every moment reached an unprecedented scale

- flourishes : The discipline of data science flourishes in this environment

- rigor : its processes must be treated with the same rigor as other components of business IT

- simultaneously : while simultaneously providing operational excellence recommendations to our customers through Azure Advisor

- the meat of the problem : a hack or a prototype can take many shortcuts and focus on “the meat of the problem” it addresses

- cut any corners : a production-ready system does not cut any corners

- plethora : This includes a plethora of concerns like architecture and design, performance, security, accessibility, telemetry, debuggability, extensibility, and so on.

- encompasses : Our definition of data engineering encompasses all of these and looks at how we can implement DevOps for data science.

- reproducible : DevOps practices allow us to build reliable, reproducible systems

- overhead : Coordination overhead means the extra effort or complexity caused by needing to manage communication, decision-making, and planning between the two teams.

- declaratively  : Vue extends standard HTML with a template syntax that allows us to declaratively describe HTML output based on JavaScript state.

- abbreviated : Single-File Component (also known as `*.vue` files, abbreviated as SFC)

- Under the hood : Under the hood, Vue compiles the templates into highly-optimized JavaScript code.

- mustaches : The double mustaches interpret the data as plain text, not HTML.

- verbose : Manually exposing state and methods via `setup()` can be verbose.
