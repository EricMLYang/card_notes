# 廣告生成拆解

## 1\.基本輸入資訊彙整

### **業主原始輸入**

- 組織

- 產品照片

- 產品描述

- 期望目標: 換季促銷、形象、新功能 … 等

- 呼籲行動

- Target：目標客群

- 文案或介紹

- 品牌資產：Logo、主色、口號

### 產品標籤

- 產品所屬產業：零售、食品、娛樂

- 產品小類：飲料、食品、遊戲 …

### 場域標籤

- 播放場域：便利商店、博物館、藥妝店…等

- 位置： 櫃檯上方、入口、產品旁邊…等

- 場域特性：

   - **便利商店**：視距近、可視時間短 → 主標 ≤7 字、單一大焦點、高對比、1 個 CTA。

   - **藥妝**：等待較久 → 可多一行教育訊息（功效/成分/使用情境），標題 + 簡短副標。

   - **販賣機**：即刻決策 → 價格 / 口味超大，按鍵/掃碼 CTA 明確，圖示化。

   - **博物館**：近距閱讀 → 字級與行距友善、可加 1 段敘述，但仍保留清楚的主視覺。
