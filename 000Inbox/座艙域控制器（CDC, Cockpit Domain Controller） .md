# **座艙域控制器（CDC, Cockpit Domain Controller）** 

- 許多人經常無法區分CDC和傳統資訊娛樂系統（Traditional IVI）之間的術語差異，實際上CDC是域控中心化架構下的一個子系統，根據定義，它的硬體系統必須整合座艙中的所有資訊娛樂功能，包括儀錶板、抬頭顯示（HUD）、語音辨識和中央資訊娛樂系統，同時，它需要相容多個虛擬機器和作業系統。目前，行業專家預估，只有不到15%的量產車輛配備了這樣的域控制器。然而，當我們將CDC與傳統的資訊娛樂系統作為一個整體討論時，它們加總在一起實際上是整個車用NAND Flash市場中最大的應用，佔據了超過80%的車用NAND Flash位元需求。在這個應用中，儲存技術從eMMC、UFS到PCIe SSD都有可能，主流容量需求從16GB到256GB，具體取決於車型的級別與應用的豐富程度。
