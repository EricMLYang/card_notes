# Analysis Basic Concepts

- 解釋變數 ( explanatory variables ) : 實驗者控制，又稱因子 ( factors )

- 反應變數 ( response variable ) : 不被實驗者控制

- 外生變項 ( extraneous variable ) : 並非現有研究中的解釋變項，但被認為影響反應變項者

- Pearson correlation coefficient: measure that quantifies the **linear** relationship between two variables ( weak 0.5, median 0.5-0.8, high 0.8)
