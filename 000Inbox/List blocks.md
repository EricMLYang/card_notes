# List blocks

## Tips

1. To create a bullet list, type `-` and then type `SpaceKey`

   - Bullet list

2. To create a numbered list, type `1.` and then type `SpaceKey`

   1. Number list

3. To create a toggle list, type `>` and then type `SpaceKey`

   + Toggle list

4. To create a todo list, type `[]` and then type `SpaceKey`

   - [x] Todo list

5. To indent a list item, type `Tab` 

6. To un-indent a list item, type `Shift` + `Tab`

7. To add style to a text, just select them and choose the style: 

   > **Bold**, *Italic*, ++Underline++, ~~Strike-through~~, `Code`, $Math$, Highlight, [Hyperlink](https://wiki.heptabase.com/)
