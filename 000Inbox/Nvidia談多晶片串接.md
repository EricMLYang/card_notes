# Nvidia談多晶片串接

Nvidia 首席技術官 Michael Kagan 分享了 Nvidia 如何透過併購 Mellanox，在加速運算和網絡連接技術上取得領先地位。他談到了從單一晶片到數百萬 GPU 集群的計算能力擴展，以及軟體在解決大規模系統挑戰方面的重要性。他還探討了人工智慧在訓練和推理方面的演化，以及未來人工智慧可能為世界帶來的變革。Nvidia 的文化是追求共贏，而非獨佔現有的利潤。透過與英特爾的合作，結合傳統計算和加速計算，可以擴大市場和服務範圍。 從單一 GPU 到數十萬 GPU 集群的擴展，需要高速網絡技術，如 Mellanox 提供的解決方案，來實現計算任務的高效分配和通訊。 隨著人工智慧技術的進步，推理工作量正在超過訓練，未來人工智慧可能會幫助我們更好地理解物理定律，甚至超越人類的能力。
