# 數據平台精要學習建議

**一、 核心基礎知識領域與學習重點**

1. **資料結構與演算法 ( Data Structures & Algorithms ):**

   - **為何重要:** 高效處理感測器數據、用戶輪廓、內容標籤；設計高效的規則匹配與內容推薦排序邏輯。JSON 本身就是一種數據結構。

   - **學習重點:** 陣列、鏈結串列、樹、圖等結構；搜尋、排序、圖形演算法；理解時間與空間複雜度。

2. **作業系統與計算機網路 (Operating Systems & Computer Networks):**

   - **為何重要:** 理解程式如何在雲端環境 (如 Azure Functions/App Service) 運行、資源如何管理、系統間如何可靠傳輸數據 (API 通信、感測器數據上傳、CMS 推播)。

   - **學習重點:** 進程/線程管理、記憶體管理；TCP/IP 協議、HTTP/RESTful API 原理、網路傳輸基本概念。

3. **資料庫系統 (Database Systems):**

   - **為何重要:** 有效儲存、查詢、管理大量配置數據、用戶輪廓、廣告內容及感測數據。

   - **學習重點:** SQL 與 NoSQL (如 Azure SQL, Cosmos DB) 的概念、Schema 設計、索引優化、事務與一致性。

4. **軟體工程與系統設計 (Software Engineering & System Design):**

   - **為何重要:** 設計模組化、可擴展、可維護的系統；理解架構模式 (如微服務、適配器模式)、API 設計原則、配置管理的重要性。

   - **學習重點:** 物件導向設計、設計模式 (GoF)、架構原則 (如關注點分離、配置驅動)、分散式系統基礎 (CAP 理論)、API 設計最佳實踐。

5. **雲端運算與大數據處理 (Cloud Computing & Big Data Processing):**

   - **為何重要:** 利用雲平台 (如 Azure) 提供的服務 (計算、儲存、無伺服器、數據處理)；處理大規模即時/批次數據 (使用 Databricks 等工具)。

   - **學習重點:** 主流雲平台核心服務 (IaaS, PaaS, SaaS)；Serverless 架構；數據管道 (ETL/ELT)、串流處理 (Streaming) 概念；熟悉 Azure 及 Databricks 主要服務。

6. **(進階) AI/機器學習 & 物聯網 (AI/ML & IoT):**

   - **為何重要:** 實現更智能的內容推薦、用戶行為預測 (AI/ML)；整合與處理多樣化的感測器數據 (IoT)。

   - **學習重點:** 機器學習基礎概念、推薦系統原理；物聯網通訊協定、感測器整合模式、邊緣運算概念。
