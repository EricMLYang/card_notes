# 多元獲利模式大全

1. **產品銷售:** 在所有產品的成本上，外加一定程度的利潤 **(** 豐田、優衣庫 )

2. **服務業銷售商品:** 服務業結合產品銷售，提高利潤 ( 全日空 )

3. **產品組合:** 搭配不同獲利率的產品，創造事業整體的利潤 ( 流通業、休閒產業 )

4. **次要產品:** 提高主要產品與合併銷售產品的獲利率 ( 餐飲業 )

5. **多元利潤設定:**  與主要產品內容相同，但視情況調整獲利率 ( 可口可樂 )

6. 事**先附加 (保險、分期) : 只銷售主要產品利潤太少，**故以售後服務增加獲取更多利潤 ( AppleCare )

7. 事**後附加（維護）: 只銷售主要產品利潤太少**，故以售後服務增加獲取更多利潤 (YANASE)

8. **服務化（顧問化）: 提供客戶運用產品時的輔助服務，藉以**創造利潤 ( IBM )

9. **次要目標: 對**主要目標客群讓利，但從其他次要顧客賺取比較多的利潤（兒童電影、自助百匯）

10. **競標：**由買方出價投標，以便用競標產品獲取更多利潤（Google Ads）

11. **動態定價**: 同一件產品的銷售金額，會根據付費者的情況而變（主題樂園）

12. **定額訂閱制：**每隔一定時間就收取固定的使用費，隨時間累積利潤 ( 索尼、賽富時）

13. **預付訂閱制：客戶預付使用費，讓利潤先落袋為安 （**報紙、雜誌）

14. **計量訂閱制：依使用量多寡收取使用費，隨時間累積利潤**（迪亞哥、AWS）

15. 回頭客：以每位使用者都會重複購買為前提來創造利潤（迪士尼樂園）

16. **長尾：透過豐富的產品線來攬客，並透過暢銷商品來攬客** （亞馬遜 商城事業）

17. **租賃：用合約幫助客戶一段時間，隨時間累積利潤（歐力士）**

18. **刮鬍刀模式：壓低商品本身的獲利，拉高附屬產品的獲利率，靠時間創造利潤（任天堂、佳能）**

19. **會員制(會費）：透過收取會費和本業利潤分進合擊，共同為企業創造利潤（**Costco）

20. **免費增值：**提供免費的產品或服務，但調高附屬品的獲利率，隨著時間創造利潤（DeNA、玩和線上娛樂）

21. **副產品（by-product）：把在事業活動衍伸得服服產品，供應給主要客戶以外的付費者（**特斯拉、JR 東京站）

22. **IP（內容）：將跨平台使用的內容或 IP 化成重要的獲利支柱（盧卡斯影業）**

23. **手續費事業：跟顧客以外的競爭者或供應商收取手續費（**樂天、亞馬遜市場）

24. **優先權：把**優先使用權，當成重要的獲利支撐（富士急樂園）

25. **三方市場：**廣告主所付的費用當成商品和服務的主要營收支柱（瑞利可）

26. **媒合：串聯供應商和使用者的對價**（Mercari）

27. **宣傳大使：大幅減免介紹人原本應付的費用，藉由攬客和培養顧客來創造利潤**（MS Office、雀巢 BARISTA）

28. **虛榮溢價：設定一群可為同樣產品支付較高金額的人（**美國運通百夫長卡）

29. **加盟：**以成功的事業手法, 授予他人使用，並當成獲利支柱（**7-Elevent）**

30. **資料存取：**將自家「累積資料」的存取權，化為重要的獲利支柱**（**Publine, 紀伊國屋書店）
