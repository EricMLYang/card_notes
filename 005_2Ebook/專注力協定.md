# 專注力協定

- 排除網路科技，例如拿走手機、減少網路…等，無法減少分心，因為人還是會用其他分心行為來代替

- 要擁有我們渴望的生活，除了要做對的事，還要排除錯的事，蛋糕對我們腰圍的影響，大於幾次的沙拉

- 步驟：

   - 認清會讓我們分心的事情，來自於內在

   - 認知到騰出時間做你真正想做的事有多重要

   - 對多餘的外在誘因進行徹底的審查，誘因不但是生產力阻力，也會降低我們幸福感

   - 協定：是一個有根據的方法，讓我們可以約束自己保持專心，讓我們真的去做我們想做的事

## 動力

- 分心的源頭是不適感和不滿足，但我們人類天生就是會容易有不適感和不滿足，我們要

- 動力的最大來源並不是追求愉悅，而是”想要排除不適感”，這才是驅動我們行為的根本原因，其他只是表面上的近因罷了

- 我們傾向回去怪罪外在的東西，嫌少去真正檢討問題的深層結構，甚至檢討自己，例如：

   -  不探討吵架真正原因，而是怪罪引起吵架的表面起火點

- 這樣我們可以把問題或責任移轉到外在因素，不用檢討自己，而分心也有類似的狀況，我們怪罪智慧手機或小孩…等，但不去承認可能自己因為工作壓力大或家庭狀況煩心很久

- 滿足感會讓生物持續追求進步的動力降低，因此滿足不持久很有可能本身就存在我們基因，我們心裡有某個部分，總是想要替心中那不停攪動的部分找出口

- 人類四種導致永不滿足的因素：

   - 無聊：人類為了避免無聊，而願意做的事情，可能超乎我們自己想像

   - 消極性偏差：比起美好經驗，壞經驗總是比較容易佔據我們心裡

   - 反芻壞經驗：我們會傾向反覆思考不好的經驗，或是很想要去做的事

   - 享樂適應：我們天生就不會讓滿足感佔據我們太久時間，再大的快樂都不會持續太久

## 壞習慣誘惑

- 非極度上癮的行為，其誘惑其實跟快要能做這件是的心情有關，例如:

   - 抽煙應該是最能被歸類為上癮的行為，但研究空服員後發現，在一段長時間的飛行，最想抽煙的時刻是在快降落的時段，因為你知道自己快要可以抽煙了，反之，整段飛行旅程不管長短，因為你知道自己不能抽，因此欲望就變得很小

- 因此，

   - 誘惑往往會被好像可以做這件事的時機或心情放大

   - 注意一些任務轉換的交換點，這時刻最容易滑手機...等

# 分心的內在狀態追蹤

- 專注力協定的建議

- 感到不適並想分心時，此時可以深刻感受一下當下體驗和心情

- 想像有一個葉子在水流上，把你內心當下的念頭放到葉子，讓他漸漸流走，你就是在旁邊看

- 提防交界點 （liminal moment），兩件事的中間通常而很容易就

   - 10分鐘法則，想做的事，沒關係，等10分鐘看看
