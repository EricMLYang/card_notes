# 01_Index

這裡放卡片索引（MOC）。

建議做法：先把卡片放進對應主題索引，再逐步補齊更細的子索引。

目前主線重點：

- `03_ai_applications_and_productization.md`
  AI 應用、Agent、evaluation、productization
- `05_data_engineering_and_analytics.md`
  Data engineering、Business Data Science、decision system
- `06_databricks_and_lakehouse_platform.md`
  Databricks、Lakehouse、semantic layer、deployment、governance
- `07_product_management.md`
  data product、產品策略、商業決策
- `10_testing_operations_and_reliability.md`
  validation、operations、guardrails、上線責任

索引原則：

- 一張卡可以出現在多個索引中，只要能幫助後續寫作與檢索
- 優先用「這張卡未來會拿來支撐什麼判斷 / 文章 / 系統設計」決定索引，不只看字面主題
- 若卡片同時涉及資料、產品、Agent、治理，優先補到主線相關索引，避免只落在單點技術索引
