# Getting More

- Getting More 是值得回味練習的好書，畢竟生活上處處有溝通價值的機會，其中基本的12個技巧是核心，可以多練習

- Top 重點

   - 目標至關重要

- 大原則

   - 做好準備: 列出清單並事先練習

   - 要因時因地制宜

   - 循序漸進為上策

   - 保持透明、建設性，但不操弄

   - 隨時溝通、開誠布公、表述願景

- 耐心理解、建立溝通橋梁

   - 要以對方為主

   - 要動之以情

   - 接納彼此差異點

   - 找出真正的問題，化問題為機會

- 重要技巧

   - 找出對方認定的標準

   - 交換評價不相等的東西

## 練習

- 目標：一定要渡過充電樁

- 對方：真正原因 ( 身體、新工作、倦怠、主管、想學新東西…等）

- 情 : 真誠的說明你是我帶過的人 top 1  喜歡的，很可惜後續五們溝通不良，我覺得我是有滿多需要改進的地方，但環境壓力大也就這樣過了

- 因時因地制宜：當對方說我恨你時，正確的回應方式是說來聽聽，可以更知道對方的感受

   - 個人感受上我覺得你滿討厭我的

- 循序 + 標換評價不相等的東西
