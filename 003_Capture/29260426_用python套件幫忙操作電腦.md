最近 Computer-Use Agent 愈來愈紅，但有一個共同痛點：要讓 Agent 真的能「看螢幕、點按鈕、敲鍵盤」，你得先準備一台沙盒環境，最好還能跨作業系統。trycua 團隊推出的 Cua 就是要解決這問題。

最大賣點是同一套 Python API 操控任何 OS，pip install cua 裝完就能截圖、點滑鼠、打鍵盤、做多點觸控手勢。

專案有四個元件：

- Cua Sandbox 是底層執行環境

- CuaBot 把 Claude Code 之類的 agent 丟進沙盒，視窗原生顯示還附 H.265 串流

- Cua-Bench 內建 OSWorld、ScreenSpot 等主流 benchmark，可匯出軌跡作強化學習

- Lume 則用 Apple Virtualization Framework 跑 VM，效能接近原生

過去做 GUI 自動化，不同平台有不同框架，現在統一成 Sandbox[.]ephemeral()，想讓 Agent 操控電腦的都值得試一輪。
