# 巴士內容播放 PoC



是否可以請 S4M 介紹其他可在公車上播放內容的廠商，

最好是關係比較好，未來可以配合 AUO 發展廣告業務的整合商。



緣由:

。S4M 目前沒有規劃車上的播放設備

。但本案跟 ABS 手上業務有綜效機會，因此 ABS 想跟車用單位繼續發展



現況角色(今日 11/21 五多方會議):

。主客戶: 總盈汽車股份有限公司，車體製造商

。螢幕供應: Adlink ( 螢幕在車內)

。車機廠商: 寶錄 ( 車機透過乙太網路傳給螢幕經緯度、站名 … 等資訊)

。缺: 其他內容供應商，幫忙播放政令文宣、可安插其他廣告內容



權責邊界:

。目前有其他案例是螢幕分割成 3 部分，

。寶錄獨立提供到站、離站資訊，車機透過乙太網路連螢幕

。總盈要安裝一個 Hub 提供網路

。內容播放商獨立播放其他螢幕



未來綜效:

。總盈是巴士製造商，PoC 成功的化未來的車子會持續搭載這個配備

。ABS 手上有巴士充電管理系統，會期望去系統上的巴士客戶持續推廣

。ABS 手上有電動貨車數據管理系統，有機會可談談看

。未來達到一定規模，可以 Join AUO 期待的廣告業務，擴大規模
