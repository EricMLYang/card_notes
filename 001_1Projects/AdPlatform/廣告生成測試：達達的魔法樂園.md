# **廣告生成測試：達達的魔法樂園**

## **科博館「達達的魔法樂園」精彩回歸 3大互動式展具沉浸式學習光電知識**

2025/08/13

新聞發布

達達虎回來囉！國立自然科學博物館與友達光電攜手打造的「達達的魔法樂園」，在歷經近兩個半月整修後，即日起重新對外開放。展區全新推出 3 個大型互動式展具，內容結合太陽能發電的綠色能源、光電原理與光譜知識，陪伴大小朋友一同開啟充滿趣味與知識性的光電科普奇幻旅程。

國立自然科學博物館館長黃文山表示，友達光電長期致力於推廣科普與永續教育，並與科博館合作多年。「達達的魔法樂園」自 2015 年起啟用以來，已累積近 200 萬參觀人次，成為中部地區極具代表性的光電科普教育平台。今（2025）年友達光電再次資助展區升級，不僅延續投入科學教育的使命，也進一步強化科博館推動永續發展與全民科普的能量。

本次更新亮點為三大互動式展具，像是能透過操作燈光控制太陽能車完成永續任務、理解光能與再生能源應用的「達達光能歷險」；以故事呈現炭火、日光燈、LED 紅綠燈與螢火蟲等常見光源背後發光原理的「發光大揭祕」；觀察不同光源光譜變化、認識光在生活場域中多元應用的「光的指紋」。所有內容皆巧妙融合太陽能、光電、光譜等科學知識，同時搭配生動有趣的情境動畫，為參觀者營造豐富的感官體驗。

除了展區更新，「達達的魔法樂園」也依據不同年齡層持續推動多元教育服務。「實境解謎」可透過分級設計的解謎任務與專人導覽，引導親子及校外教學團體探索展場知識；「達達小學堂光電課」對應 108 課綱小學三至四年級自然課程，搭配生活情境進行教學引導；「數位光學教材」提供自主學習教材，結合日常生活知識與手作練習，協助國小四到六年級學生跨地域、跨資源限制學習光電知識。

科學中心 B1「達達的魔法樂園」開展以來，以多樣化展具與童趣環境廣受好評。展場專注於光電科普教育，涵蓋立體偏光原理、光的三原色與紅外線原理等主題內容，觀眾可選擇自由參觀、預約制課程或志工團隊導覽，為學童與親子打造完整的實作式學習體驗。

歡迎大家把握暑假尾聲，一起走進充滿知識與樂趣的光電樂園，從遊戲中獲取豐富的科普知識，讓今年夏天更加充實。

## 擴散發想素材

### **1) 逆向痛點切入（Inversion for Ads）**

**忽視原因（便利商店短停留顧客）**

- 太像學校教科書 → 覺得無聊

- 字太多，顧客一眼掃過就放棄

- 與日常無關 → 無記憶點

- 太暗或太複雜 → 視覺上自動略過

**反向「救回注意力」主張**

- 「光電＝魔法」→ 趣味而非課本

- 一秒亮起角色＋發光裝置 → 馬上抓眼球

- 單一焦點，7字內 → 「會發光的樂園」

**潛在文案方向**

- 「玩到發光」

- 「一秒學會光的魔法」

- 「小孩秒懂的光電」

### **2) 動機五問（Five Whys → 承諾語）**

**親子 Why-chain**

- 為什麼要來？→ 小孩想玩 → 爸媽希望寓教於樂

- 為什麼重要？→ 希望孩子暑假/假日不只是打電動

- 為什麼還重要？→ 要找安全、有遮蔽的地方避暑

- 還擔心什麼？→ 時間太久，小孩會無聊，館內擁擠

**承諾語**

- 「15 分鐘玩出一個光電觀念」

- 「三關卡，邊玩邊懂，避暑不怕曬」

**化解障礙**

- 動線短、館內有遮蔽

- 3 座大型互動裝置 → 可見即信

### **3) 隱喻鉤子（Analogy → 視覺化）**

**隱喻方向**

- 魔術 → 光是魔法，光譜是咒語

- 電玩破關 → 3 關互動裝置＝打怪升級

- 寶藏探險 → 每一光線都是暗藏的「寶物線索」

**文案靈感**

- 「光電魔法課」

- 「挑戰三關卡，升級光之勇者」

- 「打開光的寶箱」

**第一鏡畫面**

- 小孩揮手 → 裝置瞬間發亮

- 光像魔法陣般散開

- 面板亮起 → 達達虎角色登場

## **4) 角色視角切換（Perspective → 鏡頭語言）**

- **孩子視角**：

   Hook：「這會亮！」 → 第一鏡：手一按，燈亮起，笑聲爆出

- **家長視角**：

   Hook：「玩得到，學得住」 → 第一鏡：小孩解鎖發光裝置，旁邊有知識小提示

- **老師視角**（校外教學）：

   Hook：「15 分鐘完成一課」 → 第一鏡：整班小朋友分組操作，老師輕鬆引導

- **館方視角**：

   Hook：「科學不再是課本，而是魔法」

- **看板自己**（趣味 Meta）：

   Hook：「我會發光，你敢不看嗎？」

## **5) 組合式主張（Combination Grid）**

**場景 × 益處 × 梗**

- **場景**：便利商店櫃檯螢幕／館內中庭大螢幕／電扶梯旁

- **益處**：動手玩／快速懂／親子互動／避暑有遮蔽／寓教於樂

- **梗**：開箱／闖關／升級／打怪／魔法

**快速組合範例**

- 「櫃檯螢幕 × 動手玩 × 魔法」 → 標語：手一按，光電魔法開啟

- 「館內中庭 × 親子互動 × 闖關」 → 標語：親子三關卡，挑戰光之勇者

- 「電扶梯旁 × 快速懂 × 升級」 → 標語：3 分鐘升級光電知識

- 「櫃檯螢幕 × 避暑有遮蔽 × 打怪」 → 標語：涼爽館內，打怪升級光能量

好的，這是一組根據您提供的資訊與素材，專為「親子」受眾（特別是在便利商店結帳、注意力短暫的家長）設計的廣告腳本。

**單一受眾分析：**

- **受眾：** 親子客群中的家長。

- **此刻在乎：** 快速找到一個能讓孩子放電、又有教育意義的室內週末去處。他們需要一個「有趣、不無聊、有價值」的行程。

- **溝通核心：** 強調「玩樂」與「秒懂」，而不是「學習」與「知識」。用魔法般的視覺效果，取代教科書式的沉悶感，讓家長相信「這裡很好玩，孩子會喜歡」。
